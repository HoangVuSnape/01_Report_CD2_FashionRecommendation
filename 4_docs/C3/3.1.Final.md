# 3.1 Overview of the proposed pipeline

Muc tieu: mo ta toan bo hanh trinh du lieu tu raw VCR dataset den 2 file train.txt va test.txt san sang cho mo hinh. Noi dung nay gom 3 nhom buoc: (1) raw data to sampling, (2) lam sach + N-core + mapping ID, (3) temporal split theo user.

---

## Tong quan pipeline

```
RAW DATA (VCR Dataset)
  user_activity_triplets.csv   ~64k giao dich, ~1600+ users
  picture_triplets.csv         ~50k anh (nhieu anh / outfit)
  outfits.csv                  metadata outfit
          |
          v
  BUOC 1 — slipt_10k_sample.py
          |  Loc anh chinh (displayOrder == 0)
          |  Loai outfit khong co anh hop le
          |  Sample ~10k giao dich (user co >= 3 GD)
          |
          +--> user_activity_triplets_10k.csv   ~10,000 giao dich
          +--> picture_triplets_10k.csv          2,223 anh (1/ outfit)
          +--> outfits_10k.csv                   2,223 outfits
          |
          v
  BUOC 2 — preprocess_vcr_10k.py
          |  Lam sach (loai outfit null name, loai GD mat anh)
          |  N-Core Filter: giu user co >= 5 giao dich
          |  ID Mapping: string ID -> so nguyen 0-indexed
          |
          +--> dataset_VCR_1.0_42_5.csv          9,455 GD, 553 users, 2,194 items
          |
          v
  BUOC 3 — Per-user Temporal Split (80/20)
          |  Moi user: 80% GD dau -> train, 20% GD cuoi -> test
          |
          +--> train.txt    7,350 interactions (553 users)
          +--> test.txt     2,105 interactions (553 users)
```

---

## 3.1.1 Raw data -> sampling (BUOC 1)

Dau vao gom 3 file CSV goc:

```
user_activity_triplets.csv   ~64k giao dich, ~1600+ users
picture_triplets.csv         ~50k anh (4-5 anh / outfit)
outfits.csv                  metadata outfit
```

Noi dung chinh cua BUOC 1:

1) Loc anh chinh (displayOrder = 0). Mot outfit chi giu 1 anh chinh.
2) Merge va loai outfit khong hop le (khong co anh chinh, name null).
3) Sample ~10k giao dich theo user (chi chon user co >= 3 GD) de dam bao moi user duoc chon co du lich su.

Dau ra:

| File | Rows | Ghi chu |
|---|---|---|
| user_activity_triplets_10k.csv | ~10,000 | Giup giu nguyen cot goc |
| picture_triplets_10k.csv | 2,223 | 1 anh chinh / outfit |
| outfits_10k.csv | 2,223 | Outfit co anh hop le |

---

## 3.1.2 Lam sach + N-core + ID mapping (BUOC 2)

- Lam sach: loai outfit null name va cac giao dich khong co anh hop le.
- N-core: chi giu user co toi thieu 5 giao dich.
- ID mapping: chuyen user_id va item_id thanh so nguyen 0-indexed de build adjacency.

Dau ra:

| File | Noi dung |
|---|---|
| dataset_VCR_1.0_42_5.csv | 9,455 GD, 553 users, 2,194 items |
| user_list.txt | mapping user goc -> user_id |
| item_list.txt | mapping item goc -> item_id |

Ly do items giam 2,223 -> 2,194: item chi duoc tuong tac boi user bi loai se bien mat sau N-core.

---

## 3.1.3 Per-user temporal split 80/20 (BUOC 3)

Nguyen tac:
- Moi user xuat hien trong ca train va test.
- Chia theo thu tu thoi gian, khong chia random.

Y tuong thuc thi:

```
# rank theo thoi gian cho tung user
rank = cumcount per user
train_threshold = floor(total_interactions * 0.8)
rank <= threshold -> train
rank >  threshold -> test
```

Dau ra:

| File | Users | Interactions |
|---|---|---|
| train.txt | 553 | 7,350 |
| test.txt | 553 | 2,105 |

Dinh dang file:

```
user_id item_id1 item_id2 ...
0 15 23 47 89 102 134 156 178
1 5 12 34
```
